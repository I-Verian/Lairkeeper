Thank you for using Lairkeeper! 
This is a passion project created for easier viewing of multiple accounts' lairs and stables without requiring switching between Roblox accounts. 
Please note that your dragon information is stored locally and no data is shared or saved anywhere but on your device. 
The source code is publicly available on Github at the link https://github.com/I-Verian/Lairkeeper.

When updating the app, it will automatically update to the newest version. 
Please make sure to keep the files as they are in the folder, but feel free to make a shortcut to Lairkeeper.exe for easy access!
Updating will NOT erase any of your data.
