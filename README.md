# Lairkeeper
Thank you for using Lairkeeper! This is a passion project created for easier viewing of multiple accounts' lairs and stables without requiring switching between Roblox accounts. 
Please note that your dragon information is stored locally and no data is shared or saved anywhere but on your device.
# How to use Lairkeeper?
In order to use Lairkeeper you will need a program such as 7zip or WinRar to unzip the .zip archive containing the .exe file. 
You do not have to download anything else for it to function.
